# 🏛️ Shaizullah Academy Constitution

## Mission

To become a professional Software Engineer while achieving B2 Italian through consistent learning, practical projects, and disciplined study.

---

## Vision

Build a complete academy that starts with one student and can eventually help thousands of beginners learn software engineering and Italian.

---

## Core Values

- Understand before memorizing.
- Practice before mastery.
- Build real projects.
- Learn consistently.
- Use AI as a teacher, not a replacement.
- Never stop improving.

---

## Long-Term Goal

Graduate as a confident Software Engineer with a strong portfolio and practical skills.

---

## Motto

Learn. Build. Grow.
